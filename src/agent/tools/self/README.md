# src/agent/tools/self/

Self Tools are capabilities that ROOGLE can perform directly.

## Current Self Tools (Phase 2)

- `get_help` — Explains what ROOGLE can do in plain English
- `get_balance` — Friendly balance summary (placeholder for now)
- `send_simple_message` — Prepares messages safely (demo only)
- `confirm_action` — Generates clear confirmation prompts before important steps

All tools return friendly plain-language results.

## Principle

Use a Self Tool whenever we can fulfill the user's request without needing to involve another agent.

Self tools keep the experience fast and keep the user in one simple conversation.
